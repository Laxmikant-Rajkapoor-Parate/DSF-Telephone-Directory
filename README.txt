To use Codeblocks to connect to the MySQL database
refer the following link

https://www.programmersought.com/article/88756163973/

Open Codeblocks >>Create New project>>Console Application
Project>>Build options
Search directories>>Compiler>>Add:-C:\Program Files\MySQL\MySQL server\include

Search directories>>Linker>>Add:-C:\Program Files\MySQL\MySQL server\lib



Open sqldatabase (CBP File) to open the project
